# Хуки

Папка `hooks` содержит кастомные React-хуки, которые упрощают работу с различными аспектами приложения. Каждый хук решает определённую задачу, помогая сделать код более декларативным, читаемым и удобным для переиспользования.

#### Импорт компонентов
```tsx
import { useAuth } from 'isp-ui-kit'

const { login, isLoading } = useAuth()
```

## Список хуков
