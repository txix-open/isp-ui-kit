# Components

Набор компонентов, представляющий собой обертки над Ant Design с использованием `react-hook-form`. Обеспечивает простую интеграцию форм с возможностями валидации и управления состоянием.

#### Импорт компонентов
```tsx
import { ReactJsonView } from 'isp-ui-kit'
```

## [SearchInput](?path=/docs/components-searchinput--docs)
Компонент для реализации поискового поля. Основан на Input из Ant Design с поддержкой react-hook-form. Обеспечивает быструю интеграцию фильтрации и поиска в формах.

## [ReactJsonView](?path=/docs/components-reactjsonview--docs)
Компонент для визуализации JSON-данных. Основан на библиотеке react-json-view с расширенными возможностями: кастомизация тем, управление вложенными структурами и улучшенная подсветка синтаксиса. Подходит для отладки и отображения сложных данных.
