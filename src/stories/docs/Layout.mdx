# Layout
Компоненты для построения макета страниц

#### Импорт компонентов
```tsx
import { Layout } from 'isp-ui-kit'
const { Column, NotFoundPage } = Layout
```

## Интерфейс "Три колонки"
Компоненты для построения колончатого интерфейса
### [Column](?path=/story/layout-threecolumns-column--docs)



Для корректной работы стилей внутри Column:
1. Необходимо обязательно включить CSS vars для antd config
```jsx
<ConfigProvider theme={{cssVar: true}} locale={ruRu}>
    <App/>
</ConfigProvider>
```

2. Добавить поддержку scrollbars и resize

```jsx
import 'react-resizable/css/styles.css'
import 'simplebar-react/dist/simplebar.min.css'
```

3. Обернуть приложение в Layout от antd
```jsx
import {Layout} from 'antd'

<Layout>
    <Column/>
    <div>Content</div>
</Layout>
```
