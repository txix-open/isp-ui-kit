# Utils

Папка `utils` содержит набор утилитарных функций и инструментов для упрощения различных задач в приложении

#### Импорт компонентов
```tsx
import { getConfigProperty } from 'isp-ui-kit'
```

## Список утилит
### configUtils
- **[getConfigProperty](?path=/story/utils-getconfigproperty--example)** — Хук для управления аутентификацией.

### textsUtils
- **[getText](?path=/docs/utils-textutils-описание--docs#gettext)** —
Получает текст по ключу из словаря с поддержкой fallback-значения и подстановки параметров.

- **[createTextGetter](?path=/docs/utils-textutils-описание--docs#createtextgetter)** —
Создаёт типизированную функцию `getText`, привязанную к конкретному словарю текстов.

- **[verifyTextKey](?path=/docs/utils-textutils-описание--docs#verifytextkey)** —
Проверяет наличие ключа в словаре текстов.

- **[createTextVerifier](?path=/docs/utils-textutils-описание--docs#createtextverifier)** —
Создаёт типизированную функцию для проверки существования ключей в словаре.
