# Utils

Папка `utils` содержит набор утилитарных функций и инструментов для упрощения различных задач в приложении

#### Импорт компонентов
```tsx
import { getConfigProperty } from 'isp-ui-kit'
```

## Список утилит
### configUtils
- **[getConfigProperty](?path=/story/utils-getconfigproperty--example)** — Хук для управления аутентификацией.
